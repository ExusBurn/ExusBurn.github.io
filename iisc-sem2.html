Pattern Recognition & Neural Networks

PRNN is the flagship course of the second semester — broad, rigorous, and demanding. It covers the full arc of machine learning, from classical statistical theory to cutting-edge deep learning, with heavy emphasis on implementation.

The course is structured around three major assignments. Each builds on the last, and together they represent a comprehensive survey of the field. See the dedicated Machine Learning page for a full writeup of each assignment, including classical ML from scratch, deep learning architectures, and generative models.

Parallel Programming

The Parallel Programming course is where software meets hardware. It covers three major paradigms of parallel computation, each targeting a different level of the memory hierarchy:

  • CUDA (GPU computing): Writing kernels that exploit the massive parallelism of GPU hardware. Thread blocks, warps, shared memory, memory coalescing, occupancy tuning, and the CUDA execution model. The course goes well beyond using libraries — students write their own kernels for matrix multiplication, reductions, and stencil operations.
  • MPI (distributed memory): Message Passing Interface for computations that span multiple nodes or GPUs. Point-to-point communication, collective operations (broadcast, scatter, gather, reduce, allreduce), and CUDA-aware MPI for direct GPU-to-GPU transfers without CPU staging.
  • OpenMP (shared memory): Thread-level parallelism within a single node. Pragmas, race conditions, synchronisation, and performance tradeoffs versus explicit threading.

The final project — Distributed Krylov Quantum Diagonalization — combines all three: CUDA kernels for GPU acceleration, MPI for multi-GPU distribution, and OpenMP in the CPU reference port. See the KQD page for the full writeup.

Numerical Solutions of Differential Equations

NSDE covers the classical and modern methods for numerically solving ODEs and PDEs — the mathematical backbone of simulation in physics, engineering, and scientific computing.

  • Finite Difference Methods (FDM): Discretising derivatives on a grid. Forward-time / backward-time / Crank-Nicolson schemes. Stability analysis via von Neumann stability theory. CFL conditions. Convergence rates and truncation error.
  • Finite Element Methods (FEM): Weak formulations, Galerkin projection, basis function choice (hat functions, higher-order polynomials). Assembly of stiffness and mass matrices. Application to elliptic, parabolic, and hyperbolic PDEs.
  • Spectral Methods: Fourier and Chebyshev expansions for periodic and non-periodic domains. Spectral accuracy (exponential convergence for smooth solutions). Pseudospectral methods and the FFT.
  • Krylov Iterative Solvers: Conjugate Gradient, GMRES, and their preconditioned variants for large sparse linear systems arising from PDE discretisation.

The final project uses quantum computing (Qiskit) to solve the heat equation via the LCHS algorithm — an explicit bridge between the classical numerical methods of the course and the emerging field of quantum-accelerated numerical analysis. See the LCHS page for the complete writeup.

Natural Language Processing

The NLP course covers the major methods in language understanding and generation, from classical approaches to modern large language models. Topics include tokenisation and text preprocessing, language models (n-gram through neural), word embeddings (Word2Vec, GloVe), sequence-to-sequence models, attention mechanisms, and the Transformer architecture as applied to NLP tasks.

The course provides both the theoretical grounding (information-theoretic views of language modelling, probabilistic sequence models) and hands-on experience fine-tuning pretrained models for downstream tasks.
